# go-ibc
IP Blocklist Checker (IBC) written in Go

### Prerequisites

* Go (>= 1.12)

### Installing and Updating

```
go get -u github.com/netShelter/go-ibc
```

## Using the tool

```
go-ibc --ipv4 127.0.0.1 --ipv6 ::1
```
